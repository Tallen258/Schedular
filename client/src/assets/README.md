# Assets Directory

This directory contains static resources for the application.

## Structure

- **styles/** - CSS stylesheets and styling assets
  - `aiChatResponsive.css` - Responsive styles for AI chat interface
  - `rbc-tailwind.css` - React Big Calendar customizations
  - `spinner.css` - Loading spinner styles

## What Goes Here

Place the following types of assets in this directory:

- **images/** - Image files (PNG, JPG, SVG, etc.)
- **icons/** - Icon files and icon sets
- **fonts/** - Custom font files
- **data/** - Static JSON data files
- **documents/** - PDFs and other static documents

> **Note:** This directory is for static, non-code assets only. Code files belong in other directories like `components/`, `utils/`, etc.
